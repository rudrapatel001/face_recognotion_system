Welcome to Face Recognition with Python!

This repository contains the source code for a face recognition application built using Python.

## Features
- Detect and recognize faces in images or webcam feed.
- Utilizes Python's face recognition libraries for accurate results.
- User-friendly graphical user interface (GUI) built with Tkinter.

## Usage
1. Clone this repository to your local machine.
2. Ensure you have Python installed along with the necessary libraries (requirements listed in `requirements.txt`).
3. Run the `main.py` file to launch the application.
4. Choose the option to either recognize faces in an image or via webcam feed.
5. Follow the on-screen instructions to initiate face recognition.
